Aide de la liste de diffusion %(listname)s: 

Ceci est l'aide sur les commandes par courrier pour la version
%(version)s du gestionnaire de listes "Mailman".  Vous trouverez
ci-dessous la description des commandes qui vous permettent d'obtenir
des informations sur vos abonnements et d'en modifier les options pour
les listes de ce serveur.  Chaque commande pour être placée au choix
dans l'objet ou dans le corps du message.

La plupart de ces tâches peuvent être également accomplies par
l'interface web qui se trouve à l'adresse:

    %(listinfo_url)s 

Vous pouvez notamment utiliser cette interface pour vous faire
envoyer votre mot de passe à votre adresse d'abonné. 

Les commandes relatives à une liste (s'abonner, qui y est, etc..)
doivent être envoyées à l'adresse *-request de la liste en question,
par exemple pour la liste 'mailman', utilisez 'mailman-request@...'. 

A propos des descriptions -  les mots entre "<>" sont les éléments 
OBLIGATOIRES tandis que ceux entre "[]" sont des éléments
FACULTATIFS. N'incluez pas les "<>" ou "[]" dans les commandes.

Les commandes suivantes sont reconnues: 

    subscribe [mot-de-passe] [envoi-groupé] [address=<adresse>]
        S'abonner à la liste.  Votre mot de passe est nécessaire pour
        vous désabonner ou changer vos options.  Une fois abonné à la
        liste, votre mot de passe vous sera rappelé périodiquement.
        'envoi-groupé' doit être l'une des deux valeurs: 'nodigest' ou
        'digest' (sans guillemets!).  Si vous souhaitez vous abonner
        avec une adresse autre que celle d'où vous envoyez cette
        commande, vous pouvez indiquer "address=<adresse
        électronique>" (sans parenthèses autour de l'adresse, et sans
        guillemets!)

    unsubscribe <mot-de-passe> [adresse]
		Résilier votre abonnement à la liste. Votre mot de passe doit
		correspondre à celui que vous avez fourni lorsque vous vous
		êtes abonné.  Si vous essayez de résilier votre abonnement à
		partir d'une adresse différente de celle sous laquelle vous
		êtes abonné, indiquez-la dans le champ 'adresse'.

    who
        Voir tous les abonnés à la liste de diffusion. 

    info
        Obtenir une le message introductif de cette liste. 

    lists
		Voir quelles sont les listes gérées par ce serveur Mailman.
 
    help
        Ce message.

    set <option> <on|off> <mot-de-passe> 
        Activer/désactiver des options d'abonnement. Les options
        reconnues sont:

        ack : 

		    Recevoir un accusé de réception quand vous envoyez un
		    message à la liste.

        digest : 
		    Recevoir un message quotidien regroupant les messages de
            la liste, au lieu de recevoir les messages un par un.

        plain : 
		    Recevoir les envois groupés en texte simple, non MIME
            (uniquement si le mode groupé est activé).

		nomail :
		   Cesser de recevoir les messages.  Utile si vous pensez
		   prendre quelques vacances.

        norcv : 
		    Activez ceci pour ne PAS recevoir les messages que vous
            envoyez à la liste.  Ne fonctionne pas si le mode groupé
            est activé.

        hide : 
		    Cacher votre adresse lors de la visualisation des abonnés
            de la liste.
		
		nodupes :
			Activez cette option si vous ne souhaitez pas recevoir
			plusieurs copies des messages en provenance de la liste,
			dans le cas où votre adresse a été listée dans l'un
			des champs To: ou Cc: ou que le message est destiné à
			plusieurs listes. 

    options 
		Montre les valeurs actuelles de vos options pour les listes.

    password <ancien-mot-de-passe> <nouveau-mot-de-passe>
        Changer votre mot de passe pour la liste.

    end ou --
        Arrête le traitement des commandes (utile si votre logiciel de
        messagerie ajoute automatiquement un fichier de signature -
        cela vous évitera bien des ennuis).


Les commandes doivent être envoyées à %(requestaddr)s 

Toute question ou problème à l'attention d'une personne physique doit
être envoyée à 

    %(adminaddr)s
