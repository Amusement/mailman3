La liste de diffusion `%(listname)s' vient d'être créée pour vous.
Vous trouverez ci-dessous quelques informations indispensables sur
cette liste. 

Votre mot de passe pour cette liste est: 

    %(password)s

Ce mot de passe vous est nécessaire pour configurer votre liste de
diffusion. Vous en aurez également besoin pour traiter les demandes
administratives, par exemple approuver les messages si votre liste
est modérée. 

Vous pouvez configurer votre liste de diffusion à cette adresse: 

    %(admin_url)s 

La page web pour les utilisateurs de votre liste est: 

    %(listinfo_url)s 

Vous pouvez même personnaliser ces pages web depuis la page de 
configuration de la liste.  Il est recommandé de connaître le HTML 
pour le faire.

Il y a également une interface courrier pour les utilisateurs (et
non les administrateurs) de votre liste; pour savoir comment
l'utiliser, envoyez un message ne contenant que le mot `help',
comme sujet ou dans le corps du message, à:

    %(requestaddr)s 

Pour désabonner un utilisateur : sur la page 'listinfo' de la
liste cliquez sur ou entrez l'adresse électronique de l'utilisateur
comme si vous étiez lui. Là où il indiquerait normalement son mot
de passe, entrez votre mot de passe d'administrateur. Vous pouvez
aussi utiliser votre mot de passe pour changer les préférences des
membres y compris les messages groupés, la désactivation, etc.

Pour toute question adressez-vous à mailman-owner@%(hostname)s.
