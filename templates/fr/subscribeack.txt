Bienvenue sur la liste %(real_name)s@%(host_name)s!
%(welcome)s
Pour écrire sur la liste, envoyez votre message à:

  %(emailaddr)s

Les informations générales concernant cette liste sont à l'adresse:

  %(listinfo_url)s

Si vous souhaitez vous désabonner ou modifier vos options (par
exemple activer/désactiver l'envoi groupé, changer votre mot de
passe, etc.), consultez votre page d'abonnement à l'adresse:

  %(optionsurl)s
%(umbrella)s
Vous pouvez également procéder par mail en envoyant un message à:

  %(real_name)s-request@%(host_name)s 

avec le mot `help' dans le sujet ou le corps du message (sans les
guillemets), vous recevrez alors un message avec des insctructions.

Votre mot de passe est nécessaire pour ces modifications (y compris
pour changer le mot de passe lui-même) ou pour vous désabonner.
Votre mot de passe est:

  %(password)s 

Si vous oubliez votre mot de passe, ne vous inquiétez pas, vous
recevrez un rappel mensuel vous donnant tous vos mots de passe
sur %(host_name)s, ainsi que comment vous désabonner ou changer
vos paramètres.  Il y a aussi un bouton sur votre page web
d'abonnement qui vous permet de vous reexpédier votre mot de passe
actuel.

Vous pouvez également avoir votre mot de passe expédié à votre
adresse automatiquement depuis la page mentionnée plus haut.
