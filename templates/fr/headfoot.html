Ce texte peut contenir des formats de chaînes Pythonqui sont remplacées par les attributs de la liste. La liste des substitutions valides est:
  • real_name - Le `joli' nom le la liste; habituellement le nom avec la bonne casse.
  • _internal_name - Le nom par lequel la liste est reconnue dans les URLs, où la casse est importante.
  • host_name - Le nom d'hôte pleinement qualifié du serveur sur lequel est installé le serveur de listes.
  • web_page_url - L'URL de base pour Mailman. Elle peut être concacténée avec, par exemple listinfo/%(internal_name)s pour donner la page d'infos de la liste.
  • description - La description courte de la liste de diffusion.
  • info - La description complète de la liste de diffusion.
