Cette page contient l'ensemble des requêtes administratives en attente de votre approbation pour la liste de diffusion %(listname)s . Vous trouverez d'abord la liste des demandes d'abonnement en attente (s'il y en a), suivie des éventuels messages envoyés à la liste en attente de votre approbation.

Pour chaque demande d'abonnement, choisissez Refuser ou Abonner. Si vous refusez l'abonnement, vous pouvez également donner une raison dans la zone de texte prévue à cet effet.

Pour chaque message retenu, choisir l'une des actions possibles parmi

  • Repousser - Repousse la décision à plus tard. Utilisez ce choix si vous souhaitez seulement faire suivre le message ou le conserver (voir plus bas) mais pas le traiter pour l'instant.
  • Approuver - Approuver le message et l'envoyer sur la liste.
  • Rejeter - Rejeter le message en envoyant une notification à l'expéeacute;diteur. Le message d'origine est éliminé.
  • Eliminer - Le message d'origine est éliminé sans envoyer de notification. Ce choix est utile pour les spams.

Séléctionnez l'option Conserver si vous souhaitez conserver un exemplaire pour l'administrateur du site. Ceci est utile pour les messages abusifs. Séléctionnez l'option Faire suivre à, et remplissez l'adresse où faire suivre si vous souhaitez faire suivre le message à quelqu'un qui n'est pas sur la liste.

Quand vous avez terminé, cliquez sur le bouton Soumettre les Données en haut ou en bas de la page. Ce bouton va soumettre toutes les actions sélléctionnées par toutes les requètes administratives pour lesquelles vous avez pris une décision.
