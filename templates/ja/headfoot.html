この文章には %(attribute)s の形式の文字列を 含むことができ，それはリストの属性に置きかえられます． 詳しくは， Python の文字列置き換え規則をご覧ください． 便利な属性としては，以下のようなものがあります．
  • real_name - リストの「きれいな」名前; 普通は大文字の入ったリストの名前になります．
  • _internal_name - URL の中でリストが識別される 名前．ここで，大/小文字の違いは区別されます．
  • host_name - リストサーバーが作動するホストの 正式のドメイン名(FQDN).
  • web_page_url - Mailman の基本 URL. 例えば listinfo/%(_internal_name)s を付け加えると メーリングリストの案内ページになります．
  • description - このメーリングリストの短い 説明文.
  • info - このメーリングリストの長い説明文.
