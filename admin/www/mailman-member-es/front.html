Prefacio

Resumen:

Este documento describe el interfaz del suscriptor de listas GNU Mailman 2.1. Este manual contiene instrucciones para suscripción, desuscripción, consulta de los archivos de la lista, edición de opciones de suscriptor, obtención de recordatorios de contraseñas y otras tareas a nivel del suscriptor. También responde algunas preguntas comunes de interés para usuarios de listas Mailman.
